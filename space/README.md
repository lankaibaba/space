# 订单监控看板

基于 `ordertool.py` 的可视化看板，提供本地网页展示。

## 功能特性

- **五省统一统计**: 展示湖北、湖南、新疆、河北、安徽五省的待配载订单数量和总重量
- **今日应到货**: 显示今天应到货但未签收的订单，支持省份筛选
- **明日应到货**: 显示明天应到货但未签收的订单，支持省份筛选
- **自动刷新**: 每5分钟自动刷新数据
- **手动刷新**: 提供手动刷新按钮即时更新数据

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行服务

```bash
python app.py
```

服务启动后，访问: http://localhost:5000

## 技术栈

- 后端: Flask
- 前端: HTML5 + CSS3 + JavaScript (原生)
- 数据: 订单系统 API
